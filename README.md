# Indonesian Sign Language (BISINDO) to Text Translator Using LSTM

## 🧠 Project Overview

This project focuses on developing a **real-time translator system for Indonesian Sign Language (BISINDO) into text**, with an emphasis on recognizing **basic everyday words and greetings** such as "hello" and "thank you." The goal is to improve communication accessibility for deaf and speech-impaired individuals, enabling easier interaction with the broader community.

## 🎯 Objectives

- Bridge communication gaps between people with hearing or speech disabilities and non-signers.
- Accurately recognize and translate **basic BISINDO gestures** in real-time.
- Lay a foundation for expanding to more complex vocabulary and phrases in the future.

## 🔍 Current Focus and Approach

- Concentrating on a **limited vocabulary of fundamental signs**, prioritizing common and essential words.
- Developing a system that can effectively distinguish simple dynamic and static gestures.
- Preparing groundwork for future inclusion of more advanced and continuous sign language recognition.

## 🚧 Challenges and Future Work

- **Gesture similarity** remains a challenge, especially for signs that look alike.
- Expansion of the **dataset diversity and size** is necessary to improve model robustness.
- Exploring advanced deep learning architectures to enhance accuracy and speed.

## 🌍 Impact

By targeting essential signs first, this project aims to provide immediate practical benefits for daily communication and serves as a stepping stone toward comprehensive BISINDO translation systems.
